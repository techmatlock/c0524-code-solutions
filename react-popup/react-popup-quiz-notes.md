# typescript-popup-quiz-notes

## Quiz Questions

Answer the following questions in the provided markdown file before turning in this exercise:

- What is a React Portal?

An API that lets you render some children into a different part of the DOM.

- Where does a Portal get rendered in the DOM?

Into the domNode parameter that you specify.

- When might you want to use a Portal?

When a React parent component has a hidden value of overflow property, and we need a child component to openly come out of the current tree hierarchy. Dialogs and modals.
